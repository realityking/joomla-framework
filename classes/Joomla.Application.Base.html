Parameters

$input

\Joomla\Input\Input

An optional argument to provide dependency injection for the application's input object. If the argument is a InputCli object that object will become the application's input object, otherwise a default input object is created.

$config

\Joomla\Registry\Registry

An optional argument to provide dependency injection for the application's config object. If the argument is a Registry object that object will become the application's config object, otherwise a default config object is created.
