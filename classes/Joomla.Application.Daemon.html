Class to turn Cli applications into daemons.

It requires CLI and PCNTL support built into PHP.

see \Joomla\Application\http://www.php.net/manual/en/book.pcntl.php
see \Joomla\Application\http://php.net/manual/en/features.commandline.php
since 1.0

Parameters

$input

\Joomla\Input\Cli

An optional argument to provide dependency injection for the application's input object. If the argument is a InputCli object that object will become the application's input object, otherwise a default input object is created.

$config

\Joomla\Registry\Registry

An optional argument to provide dependency injection for the application's config object. If the argument is a Registry object that object will become the application's config object, otherwise a default config object is created.

Returns

integerOn success, the PID of the child process is returned in the parent's thread of execution, and a 0 is returned in the child's thread of execution. On failure, a -1 will be returned in the parent's context, no child process will be created, and a PHP error is raised.
